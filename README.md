## ‚ú¶ Usage Guide

### Starting a New Engagement

1. **Open** `bug-bounty-tracker.html` in your browser
2. **Click** `Target Info` in the sidebar ‚Üí fill in domain, program, platform
3. **Start** with the **Quick Checklist** tab for fast initial coverage
4. **Work through** Priority 1 ‚Üí Priority 2 ‚Üí Advanced as the engagement deepens
5. **Export** your findings report at any time from the Reports view

### Exporting Results

| Format | Best For |
|--------|----------|
| ūüďč **Copy to Clipboard** | Pasting into Notion, Obsidian, or notes |
| ūüďĄ **Export TXT** | Plain-text report for sharing |
| ūüďä **Export CSV** | Importing into Excel / Google Sheets |
| ūüĖ®ÔłŹ **Print** | PDF generation via browser print dialog |

### Resetting for a New Target

Click **Reset Progress** in the sidebar. This clears all checkboxes and the activity log while **preserving** your Target Info and Workflow state.
